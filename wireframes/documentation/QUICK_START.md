## Overview
This guide provides a quick reference for using the detailed Balsamiq wireframes designed specifically for review scraping applications with efficiency, minimalism, and purposeful functionality.

## Installation
1. Download and install [Balsamiq Mockups 3](https://balsamiq.com/wireframes/desktop/) or use [Balsamiq Cloud](https://balsamiq.com/cloud/)
2. Import the `.bmml` files from this folder

## Detailed Wireframe Files

| File Name | Purpose | Key Features |
|-----------|---------|--------------|
| `dashboard-overview.bmml` | Main dashboard | Quick stats cards, recent activity table, key metrics |
| `scraper-management.bmml` | Scraper management | Filter bar, bulk operations, data table, pagination |
| `review-data-viewer.bmml` | Review browsing | Star ratings, platform icons, export options, review cards |
| `excel-export-modal.bmml` | Export configuration | Platform selection, date range, column customization |
| `scraper-configuration-modal.bmml` | Scraper setup | Platform dropdown, URL input, scheduling, advanced settings |

## Design System

### Color Palette
- **Primary Actions**: Blue (#2563EB) - Export buttons, primary CTAs
- **Secondary Elements**: Gray (#6B7280) - Secondary buttons, disabled states
- **Success States**: Green (#10B981) - Active scrapers, successful operations
- **Warning States**: Yellow (#F59E0B) - Pending operations, warnings
- **Error States**: Red (#EF4444) - Failed scrapers, errors

### Typography & Spacing
- Consistent font sizing for headers, body text, and labels
- Optimized line heights for data table readability
- Systematic spacing between elements for visual clarity
- Left-aligned text for data tables, centered for actions

## Core Design Principles

### Efficiency
- Optimized for data-heavy interfaces with minimal cognitive load
- Quick access to key metrics and common actions
- Bulk operations for managing multiple items

### Minimalism
- Clean, uncluttered layouts focusing on essential functionality
- Progressive disclosure of information
- Reduced visual noise for better user focus

### Consistency
- Uniform navigation patterns across all screens
- Standardized button styles and interactions
- Consistent visual hierarchy and spacing

### Purposeful Design
- Every element serves the core functionality of scraping and data gathering
- Action-oriented layouts with prominent CTAs
- Clear visual indicators for status and actions

## Customization Guidelines

### Brand Adaptation
- Replace "Review Scraper" with your application name
- Adjust color palette to match brand guidelines
- Modify platform icons to match your supported review sites
- Customize terminology based on your industry

### Feature Extensions
- Add new platforms to scraper configuration
- Include additional export formats beyond Excel
- Implement advanced analytics dashboards
- Add user management and permissions

## Export Options
- **PDF Export**: Use Balsamiq's built-in PDF export for documentation
- **PNG Export**: Export individual screens for presentations
- **Development Reference**: Use as reference for frontend development

## Troubleshooting
- If files won't import, ensure you're using Balsamiq Mockups 3 or later
- Try importing one file at a time for large projects
- Check file extensions are correct (`.bmml` not `.bmpr`)
- For compatibility issues, refer to README.md for detailed instructions
- View VISUAL_PREVIEW.md for detailed layout descriptions
